Sentiment Analyzer (AI Project)

Description
A simple AI-based sentiment analysis application that classifies user input text as Positive, Negative, or Neutral using Natural Language Processing (NLP).

 Features
- Real-time sentiment analysis
- Command-line interface
- Positive / Negative / Neutral classification
- Exit command

Technologies Used
- Python
- TextBlob (NLP library)
